# Planckd2025-HackyGo

## Track
Quantum Machine Learning (QML)

## Problem Statement
MNIST classification using classical, quantum, and hybrid models.

## Team
- Team Name: HackyGo  
- Member: Atharva Sanjay Sakhare (@Athycodz)

## Project Overview
We implemented:
1. **Classical SVM** — baseline accuracy ≈ 0.87  
2. **Hybrid Quantum Classifier** (PennyLane + PyTorch backend) — accuracy ≈ 0.92  

The hybrid model uses:
- PCA feature compression → 6 principal components  
- AngleEmbedding + StronglyEntanglingLayers on 6 qubits  
- Adam optimizer, 25 epochs
